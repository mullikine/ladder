This ladder is based on a simplified version of the ELO rating system, like on the MSN Gaming Zone. It is calculated as follows:

ΔRdifference value of rating, each winner/loser gains/loses
RWwinner's current rating (in team, sum of team's rating)
RLloser's current rating (in team, sum of team's rating)
Nnumber of players
ΔR= 16 + (RL - RW) x 2 / N x (16 / 400)  if (ΔR > 31) ΔR = 31;
if (ΔR < 1) ΔR = 1;
    Example:
  • 1vs1
    winner: 1700;  loser: 1600
    ↓   ΔR = 16 + (1600 - 1700) x (16 / 400) = 12
    winner: 1712, loser: 1588
  • 3vs3
    winners: 1500, 1600, 1700;  losers: 1700, 1800, 1900
    ↓   ΔR = 16 + (5400 - 4800) x 2 / 6 x (16 / 400) = 24
    winner: 1524, 1624, 1724; loser: 1676, 1776, 1876
